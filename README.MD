# Projeto Modulo: Logica de Programação - ADATech

Esse tem como objetivo criar um jogo da forca com os conhecimentos adquiridos no curso de logica de programação com c#

## Como Jogar

1- Rode o projeto

2- O programa irá pedir para que escolha uma opção, jorgar sozinho ou jogar em dupla

3- apos escolher a opção você terá que advinhar qual é a palavra com as dicas que o jogo irá lhe informar que são	
	a quantidade de letras e a categoria da palavra
	
## Requisitos para Executar

- É necessario ter instalado o runtime ou o SDK do .NET 8
